# E5 — 端侧验证

## 1. 实验定位

本实验对应 [实验方案.md](/D:/EdgeKVLife/论文规划/实验方案.md:305) 的第 `6` 部分，对应主规划中的部署价值验证。

## 2. 回答什么 / 卡住后续什么

本实验要回答：在 Jetson 或受限显存 GPU 上，策略层是否仍然带来真实收益，且没有被自身开销吞噬。

## 3. 前置依赖

1. `H3` 已证明可接入
2. EdgeKVTiers 原型可在真实引擎运行

## 4. 使用环境与资源

1. 平台：`Jetson Orin Nano 16GB` 或同级边缘 GPU；若硬件暂不可用，使用服务器 GPU 的受限显存配置作为替代，但论文中必须明确替代性质。
2. 模型：优先使用 `Qwen2.5-7B-Instruct` 的 4-bit 端侧可部署版本；若端侧无法承载，使用同系列更小模型做趋势验证，并保留服务器侧同模型对照。
3. 推理系统：端侧运行 `vLLM + LMCache + EdgeKVTiers`；记录是否启用 prefix caching、offload 路径、量化/稀疏化 kernel 以及模块开关。
4. 机制层：`static-int4` 使用 `KIVI/KVQuant` 或等价实现，`Full` 使用完整 `COP/TMS/LPE/RRS`，默认系统使用 `vLLM default`。
5. 负载：缩短后的 `ShareGPT` 100 会话 trace，必要时加入小规模 RAG chunk 复用 trace；trace 结构要与服务器主实验一致。
6. 质量评测：沿用 `H4/E2` 的 `LongBench` 质量损失定义；端侧可只跑代表性子集，但仍需报告 `QLoss`。
7. 硬件记录：记录设备型号、可用显存、CPU/内存、存储介质、功耗采集方式、温控状态和带宽估计；可选记录 `tegrastats` 能耗。
8. 对比：`Full`、`vLLM default`、`static-int4` 使用相同模型、trace、显存预算和并发设置。

## 5. 实验执行步骤

1. 将原型移植到端侧。
2. 回放与服务器相同结构的 trace。
3. 记录 `TTFT_p95`、`QLoss`、`M_peak`、`T_policy`、`c_mig`。
4. 与服务器侧相同预算档结果对照趋势。

## 5.1 对应伪代码

### 端侧运行入口

```python
def run_e5_edge(trace, edge_device, policies):
    rows = []
    for policy in policies:
        deploy(policy, device=edge_device)
        metrics, trace_path = replay_on_device(trace, device=edge_device, policy=policy)
        rows.append({"policy": policy, **metrics, "trace_path": trace_path})
    return rows
```

### 服务器-端侧趋势对照

```python
def compare_edge_server(edge_rows, server_rows):
    joined = []
    for edge in edge_rows:
        server = next(r for r in server_rows if r["policy"] == edge["policy"])
        joined.append({
            "policy": edge["policy"],
            "edge_TTFT_p95": edge["TTFT_p95"],
            "server_TTFT_p95": server["TTFT_p95"],
            "edge_QLoss": edge["QLoss"],
            "server_QLoss": server["QLoss"],
            "edge_T_policy": edge["T_policy"],
            "edge_c_mig": edge["c_mig"],
        })
    return joined
```

### 端侧开销门槛检查

```python
def check_e5_success(rows):
    full = next(r for r in rows if r["policy"] == "Full")
    default = next(r for r in rows if r["policy"] == "vLLM default")
    return (
        full["TTFT_p95"] < default["TTFT_p95"] and
        full["T_policy"] + full["c_mig"] < (default["TTFT_p95"] - full["TTFT_p95"])
    )
```
