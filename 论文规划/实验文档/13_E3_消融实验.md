# E3 — 消融实验

## 1. 实验定位

本实验对应 [实验方案.md](/D:/EdgeKVLife/论文规划/实验方案.md:262) 的第 `4` 部分，用于验证每个模块的独立贡献。

## 2. 回答什么 / 卡住后续什么

本实验要回答：去掉任一模块后，核心指标是否出现可解释退化。

## 3. 前置依赖

1. `E2` 已产出代表性 workload 与预算组合
2. 四模块支持配置开关

## 3.1 实验所需环境与资源

1. 模型：沿用 `E2` 的主模型 `Qwen2.5-7B-Instruct`；如 `E2` 已补第二模型，本实验优先选择主模型代表性 cell，不强制扩到全模型。
2. 推理系统：使用 `E2` 同一套 `vLLM + LMCache + EdgeKVTiers` 原型，确保模块开关只改变目标模块，不改变底层引擎或 trace。
3. 数据与负载：从 `E2` 选择一个代表性 workload，优先选择 `ShareGPT` 中等 `M_budget`、中等 `epsilon` 的 cell；必要时再补 RAG 混合 cell。
4. 质量与三元组：沿用 `H4/E2` 的 `(alpha,beta,ell)` 标定表和 `LongBench` 质量损失定义。
5. 硬件：与 `E2` 主结果相同的服务器 GPU；若 `E5` 端侧结果出现异常，可在端侧追加小规模消融。
6. 日志资产：必须保存每个消融配置的完整 JSONL trace，字段包含模块开关、迁移动作、驱逐原因、RRS 动作、`T_policy` 和 `QLoss`。

## 4. 消融配置

1. `Full`
2. `-TMS`
3. `-LPE`
4. `-RRS`
5. `-COP`
6. `-quality`

## 5. 实验执行步骤

1. 从 `E2` 选一个最有代表性的 workload 和预算档。
2. 对上述 6 种配置分别运行。
3. 记录 `TTFT_p95`、`QLoss`、`N_mig`。
4. 对比每个配置相对 `Full` 的退化幅度。

## 5.1 对应伪代码

### 消融运行入口

```python
ABLATIONS = {
    "Full":      {"use_tms": True,  "use_lpe": True,  "use_rrs": True,  "use_cop": True,  "use_quality": True},
    "-TMS":      {"use_tms": False, "use_lpe": True,  "use_rrs": True,  "use_cop": True,  "use_quality": True},
    "-LPE":      {"use_tms": True,  "use_lpe": False, "use_rrs": True,  "use_cop": True,  "use_quality": True},
    "-RRS":      {"use_tms": True,  "use_lpe": True,  "use_rrs": False, "use_cop": True,  "use_quality": True},
    "-COP":      {"use_tms": True,  "use_lpe": True,  "use_rrs": True,  "use_cop": False, "use_quality": True},
    "-quality":  {"use_tms": False, "use_lpe": True,  "use_rrs": True,  "use_cop": True,  "use_quality": False},
}

def run_e3(rep_cell, trace):
    rows = []
    for name, flags in ABLATIONS.items():
        runner = build_runner("Full", **flags, M_budget=rep_cell["M_budget"], epsilon=rep_cell["epsilon"])
        metrics, trace_path = runner.replay(trace)
        rows.append({"ablation": name, **metrics, "trace_path": trace_path})
    return rows
```

### 退化幅度计算

```python
def summarize_ablation(rows):
    full = next(r for r in rows if r["ablation"] == "Full")
    summary = []
    for r in rows:
        summary.append({
            "ablation": r["ablation"],
            "delta_p95": r["TTFT_p95"] - full["TTFT_p95"],
            "delta_qloss": r["QLoss"] - full["QLoss"],
            "delta_mig": r["N_mig"] - full["N_mig"],
        })
    return summary
```

### 典型错误样本抽取

```python
def extract_top_failures(trace_path, k=3):
    events = load_trace_events(trace_path)
    ranked = sorted(events, key=lambda e: e["t_ttft_ms"], reverse=True)
    return ranked[:k]
```
