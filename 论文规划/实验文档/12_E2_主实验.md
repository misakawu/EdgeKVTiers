## 2. 回答什么 / 卡住后续什么

本实验要回答：

1. `TMS` 二维联合调度是否整体优于强基线
2. 在线方法是否在同 `(M_budget, epsilon)` 下形成 Pareto 优势
3. 在线方法与离线上界的差距是否可解释

2. 四模块已完成构筑
3. 真实引擎原型可稳定运行

## 4. 使用环境与资源

1. 模型：主实验以 `Qwen2.5-7B-Instruct` 为主；正式论文结果应至少补一个 `Llama-3.1-8B-Instruct` 或 `DeepSeek-R1-Distill-Qwen-7B` 的代表性 cell，证明方法不依赖单模型。
2. 推理系统：`vLLM + LMCache` 作为真实引擎环境，开启 prefix caching、cache/offload 钩子和统一 trace 日志；解析仿真器只用于离线上界和大矩阵辅助验证。
3. 量化/稀疏化机制：`KIVI/KVQuant` 提供 `static-int4/int8` 与动态降级，`H2O/SnapKV` 提供 `static-sparse-k` 与 `sparse_k` 档。
4. 负载：`ShareGPT` 多会话负载和 `RAG chunk` 混合负载，两类 workload 都要覆盖；每条 trace 固定版本、固定随机种子。
5. 质量数据：`LongBench` 子集或等价任务质量评测用于计算 `QLoss`，所有系统收益必须同时报告质量预算合规情况。
6. 预算：`3 x 3` 的 `(M_budget, epsilon)`，显存预算建议覆盖紧/中/松三档，质量预算覆盖紧/中/松三档。
7. 基线环境：`vLLM default`、`LRU/LFU`、`RAGCache/PGDSF` 等价实现、`static-int4`、`static-sparse-k`、`LPE-only`、`Belady`、`Pareto-OPT` 使用同一模型、trace 和预算。
8. 重复与硬件：每个 cell 5 次；记录 GPU 型号、显存限制方式、CPU/SSD offload 路径、CUDA/driver、vLLM/LMCache/KIVI/H2O 版本。

## 6.1 对应伪代码

### 主实验运行入口

```python
def run_e2(matrix_cfg, workloads, baselines, seeds):
    results = []
    for workload in workloads:
        trace = load_trace(workload)
        for M_budget in matrix_cfg["M_budget"]:
            for epsilon in matrix_cfg["epsilon"]:
                for policy in baselines:
                    for seed in seeds:
                        set_seed(seed)
                        runner = build_runner(policy, M_budget=M_budget, epsilon=epsilon)
                        metrics, trace_path = runner.replay(trace)
                        results.append({
                            "workload": workload,
                            "M_budget": M_budget,
                            "epsilon": epsilon,
                            "policy": policy,
                            "seed": seed,
                            **metrics,
                            "trace_path": trace_path,
                        })
    return results
```

### 指标聚合与主结果矩阵

```python
def aggregate_e2(results):
    grouped = groupby(results, ["workload", "M_budget", "epsilon", "policy"])
    table = []
    for key, rows in grouped.items():
        table.append({
            **unpack_key(key),
            "TTFT_p95_mean": mean(r["TTFT_p95"] for r in rows),
            "TTFT_p95_std": std(r["TTFT_p95"] for r in rows),
            "QLoss_mean": mean(r["QLoss"] for r in rows),
            "hit_mean": mean(r["hit_rate"] for r in rows),
            "M_peak_mean": mean(r["M_peak"] for r in rows),
            "N_mig_mean": mean(r["N_mig"] for r in rows),
            "T_policy_mean": mean(r["T_policy"] for r in rows),
        })
    return table
```

### Pareto 前沿与门槛检查

```python
def check_e2_success(table):
    passed = 0
    total = 0
    for cell in cells(table):
        tms = pick(cell, "Full")
        competitors = [pick(cell, p) for p in ["RAGCache", "static-int4", "LPE-only"]]
        best_other = min(competitors, key=lambda x: x["TTFT_p95_mean"])
        total += 1
        if tms["QLoss_mean"] <= cell["epsilon"] and tms["TTFT_p95_mean"] < best_other["TTFT_p95_mean"]:
            passed += 1
    return passed, total
```
